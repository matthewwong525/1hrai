# First 90 Days — Action Plan

Adapted from the generic 90-day playbook for Matt's situation: niche already chosen (AI for small business), recording setup exists (OBS + screen recording), pipeline being built in public.

## Week 1: Foundation

- [ ] Finalize channel branding: name, description, banner (2560x1440), profile pic (800x800)
- [ ] Set up recording space (already in progress — see `02-recording/`)
- [ ] Install vidIQ or TubeBuddy (free tiers) for keyword research
- [ ] Create Canva account for thumbnails (until automated pipeline is built)
- [ ] Establish the "Day X" thumbnail template — consistent look from video #1
- [ ] Set up email sign-up with lead magnet (e.g., "AI Tools Cheat Sheet for Small Business")

## Weeks 2-4: First Videos

- [ ] Research 10 video ideas using keyword tools (see `01-ideation/`)
- [ ] Script hooks for each video (full script for first 30 seconds, outline for body)
- [ ] Record and publish 4 videos by end of month 1
- [ ] Create custom thumbnails for every video — no auto-generated
- [ ] Write SEO-optimized titles and descriptions (see `06-publishing/`)
- [ ] Share in relevant communities: Reddit (r/smallbusiness, r/artificial), LinkedIn
- [ ] Add affiliate links for any AI tools demoed

## Weeks 5-8: Build Rhythm

- [ ] Lock in weekly upload schedule (1 long-form + 3 Shorts minimum)
- [ ] Start posting YouTube Shorts — extract from long-form recordings (see `04-clips/`)
- [ ] Study retention graphs for every video — note every drop-off point
- [ ] A/B test 2 different thumbnail styles using YouTube's Test & Compare
- [ ] Respond to every single comment within 24 hours
- [ ] Create first playlist (e.g., "Building a YouTube Channel with AI")
- [ ] Add end screens to all videos
- [ ] Goal: 8 more videos = 12 total

## Weeks 9-12: Optimize & Iterate

- [ ] Review analytics: which topics got the best retention? Double down.
- [ ] Improve hooks — test different opening styles based on retention data
- [ ] Try batch recording (2-3 videos in one session with different topics)
- [ ] Build a 2-video buffer to prevent gaps
- [ ] Start building automation pipeline components (editing, thumbnails)
- [ ] Join creator communities in AI/tech niche
- [ ] Goal: 12 more videos = 24 total, 3+ Shorts/week

## 90-Day Benchmarks

Don't panic if numbers are low. Focus on improvement trajectory, not absolutes:
- Published 20-25+ videos
- Identified your 3 best-performing topics
- Developed a consistent upload rhythm
- Have a recognizable "Day X" thumbnail style
- Understand your retention patterns
- Reply to comments within 24 hours
- Email list started (even if small)

## Beyond 90 Days

- Apply for YPP at 500 subs (Tier 1: memberships/Super Chat) and 1,000 subs (Tier 2: ad revenue)
- Start exploring collaborations with similar-size creators
- Begin creating digital products based on audience pain points
- Scale Shorts output as clip extraction automation comes online
- Continue the analytics -> ideation feedback loop
