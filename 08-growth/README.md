# 08 — Growth & Distribution

## Goal
Expand reach beyond YouTube through cross-posting, community engagement, and SEO optimization.

## Current Status
Not started

## System Spec
- Cross-post clips to Twitter/X, LinkedIn, TikTok, and Instagram Reels
- Generate platform-specific captions and hashtags for each clip
- Monitor and respond to YouTube comments (flag questions for Matthew, auto-heart positive comments)
- SEO: optimize video descriptions, tags, and titles based on search trends
- Repurpose transcripts into blog posts or newsletter content
- Track cross-platform performance to identify best distribution channels

## Tools/APIs Needed
- Twitter/X API (post clips, engage)
- LinkedIn API (post content)
- TikTok API (upload clips)
- YouTube Data API (comment management)
- Claude API (caption/content generation, comment drafting)
